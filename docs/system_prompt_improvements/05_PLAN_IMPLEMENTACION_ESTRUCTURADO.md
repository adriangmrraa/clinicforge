## ⚠️ CRITERIOS DE APROBACIÓN

### **Para cada hito se requiere:**
1. ✅ **Checklist completo** - Todas las tareas del hito
2. ✅ **Verificación exitosa** - Tests pasan
3. ✅ **Sin regresiones** - Funcionalidad existente intacta
4. ✅ **Aprobación explícita** - Usuario dice "APROBAR" o similar

### **Condiciones para rechazar hito:**
1. ❌ **Errores de sintaxis** en el código
2. ❌ **Tests fallan** en verificación
3. ❌ **Regresiones detectadas** en funcionalidad
4. ❌ **Usuario dice "RECHAZAR"** o similar
